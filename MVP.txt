Pitch: a social media website where you can post what current albums/music
that you are listening to and be able to post comments about that album.
user will be able to share which song is their favorite and least favorite
or any metaphysical speculations.

MVP features:
1 - User will be able to find an album and post on this website.
2 - will connect to the spotify API to get data to show onscreen.
3 - User can post their own comments about said album


Future features:
1 - Users will be able to add friends and see what they are listening to
2 - User will be able to like/dislike other users ratings
3 - User will be able to see how long they have been listening to music
